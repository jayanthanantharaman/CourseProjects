Deliverable contains the following:

1.MatrixMul folder which has the source code for the project.
2.MatMulDriver.jar which cah be directly used on the cluster to verify the results
3.sample_input.csv is used as input to the MapReduce program.
4.part-r-00000 - Results of the project
5.matrixmul.sh contains the command to run the jar on cluster.



Project contains application.properties which is used to read configurable values like the Matrix dimensions or localpath values.
Currently- MatMulDriver.jar is configured to give results for 10X10 10X5 matrix. 
Matrix dimensions can be changed appropriately get results for other matrix multiplication inputs.
